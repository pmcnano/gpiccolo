# gpiccolo.com

Personal portfolio and resume site for Paul Gonzalez.

**Stack:** Astro · Plain CSS · Cloudflare Pages  
**Languages:** English (`/`) · Spanish (`/es/`)  
**Themes:** Terminal dark (default) · Clean light

## Development

```bash
npm install
npm run dev      # http://localhost:4321
```

## Deployment

Deployed automatically via Cloudflare Pages on push to `main`.

- Build command: `npm run build`
- Output directory: `dist`
- Node version: `22`
